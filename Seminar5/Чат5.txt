Для получения текста из тега можно использовать свойство innerHTML элемента. Например, чтобы получить текст из тега DIV, можно использовать следующий код:

```
<div id="example">Пример текста</div>

<script>
  var text = document.getElementById("example").innerHTML;
  console.log(text); // Выведет "Пример текста"
</script>
```

Аналогично можно получать текст из любых других тегов. Главное, чтобы у элемента был уникальный идентификатор (атрибут id).
