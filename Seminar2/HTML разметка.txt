Создать сайт (html-документ с названием index html), с рассказом о чём угодно Например, о себе или о любимом коте 

Страница должна содержать как минимум:

1 Два заголовка - <h1>Я заголовок истории.</h1>
Имеется шесть элементов заголовка: <h1> (en-US), <h2> (en-US), <h3> (en-US), <h4> (en-US), <h5> (en-US) и <h6> (en-US). Каждый элемент представляет разный уровень контента в документе; <h1> представляет главный заголовок, <h2> представляет подзаголовки, <h3> представляет под-подзаголовки и так далее.

2 Два абзаца (параграфа) - <p>Я параграф, да, это я </p>

3 Одну картинку из интернета, которая находится в свободном доступе, то есть которая будет доступна без регстранции и смс ;) 
<p>Я создал ссылку на
  <a href="https://www.mozilla.org/ru/"
     title="Лучшее место для поиска дополнительной информации
     о миссии Mozilla и о том, как внести свой вклад">домашнюю страницу Mozilla
  </a>.
</p>
описание появится, если навести курсор на ссылку

4 Одну картинку, которая будет браться локально Все локальные картинки должны храниться в папке img, которая должна лежать рядом с html-документом 
<img src="images/dinosaur.jpg" alt="Голова и туловище скелета динозавра; у него большая голова с длинными острыми зубами">

5 Один нумерованный список со значениями 
<ol>
 <li>Текст Первый пункт</li>
 <li>Текст Второй пункт</li>
 <li>Текст Третий пункт</li>
</ol>
Если не указывать никаких дополнительных атрибутов и просто написать тег <ol>, то по умолчанию применяется список с арабскими числами (1, 2, 3,...),

6 Один маркированный список со значениями 
<ul>
 <li>Текст Первый пункт</li>
 <li>Текст Второй пункт</li>
 <li>Текст Третий пункт</li>
</ul>
В списке непременно должен присутствовать закрывающий тег </ul>, иначе возникнет ошибка. Закрывающий тег </li> хотя и не обязателен, но советуем всегда его добавлять, чтобы четко разделять элементы списка.

7 Одну ссылку 
<p>Я создал ссылку на
  <a href="https://www.mozilla.org/ru/"
     title="Лучшее место для поиска дополнительной информации
     о миссии Mozilla и о том, как внести свой вклад">домашнюю страницу Mozilla
  </a>.
</p>
описание появится, если навести курсор на ссылку

8 Один локальный, подключенный css-файл, в котором прописаны пара стилей, применяемых на странице 
<link rel="stylesheet" href="styles.css">

Примечание:
html-документ не должен содержать ошибок при проверке в https://validator w3 org/#validate_by_input
